Lancer l'audit Puppeteer

Le script tourne en local sur ton poste. Il va auditer toutes les pages du site via Chrome headless et envoyer les résultats ici automatiquement.

  1. Ouvre un terminal dans le dossier GestionnaireDeSite
  2. Connecte-toi (variable d'environnement GDS_TOKEN optionnelle)
  3. Lance la commande ci-dessous
node scripts/puppeteer-audit.js

L'audit dure ~5-10 minutes selon le nombre de pages. Les résultats s'affichent ici dès la fin.
